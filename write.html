Make.com — marking an article Done

After module 5 (WordPress POST), add an HTTP module → Make a request:

Method: PATCH
URL:
https://firestore.googleapis.com/v1/projects/ugtraveller-writers/databases/(default)/documents/articles/{{1.queue_id}}?key=AIzaSyANyqDudAYNjZACFNlbwMVUlVVwrEKNNcA

Header: Content-Type: application/json

Body (raw JSON):
{
  "fields": {
    "status":     {"stringValue": "done"},
    "postId":     {"stringValue": "{{5.data.id}}"},
    "articleUrl": {"stringValue": "{{5.data.link}}"},
    "doneAt":     {"stringValue": "{{formatDate(now; \"YYYY-MM-DDTHH:mm:ssZ\")}}"}
  }
}

The status flips to Done and the View Article button appears in realtime — no page open required.
